# 🌸 Rise & Bloom MTL

> Un espace pour grandir, briller et se connecter.
> *A warm, girly & seriously impactful space for ambitious women in Montréal.*

## About

**Rise & Bloom MTL** is a community for ambitious women in Montréal who want to grow — personally and professionally. We host intimate workshops in warm, cozy cafés across the city, bringing together women for career coaching, personal development, mentorship, and genuine connection.

Built with love in Montréal, Québec 🌷

- **Google Fonts** — Playfair Display + DM Sans
- **No frameworks. No dependencies. Just clean code.**

## Deployment — GitHub Pages

This site is deployed via **GitHub Pages** from the `/docs` folder.

To update:
```bash
# Edit index.html at root, then sync to docs/
cp index.html docs/index.html
git add .
git commit -m "Update website"
git push
```
